# react-native-simple-masonry-list

simple masonry list for react native

## Installation

```sh
npm install react-native-simple-masonry-list
```

## Usage

```js
import { multiply } from "react-native-simple-masonry-list";

// ...

const result = await multiply(3, 7);
```
